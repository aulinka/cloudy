## To start application
``` docker-compose up```
- potrebne si vytvorit uzivatela pomocou **administration console** na adrese [keycloak:8080](http://keycloak:8080/auth/) s prihlasovacimi udajmi **keycloak** (username, password)
- v sekcii *Manage* -> *Users* -> *Add user*, vyplnit len username a ulozit
- v zalozke **Credentials** nastavit heslo a ulozit
- prejst na adresu [localhost](http://localhost) a prihlasit sa pomocou prihlasovacich udajov, nasledne je aplikacia pripravena na pouzivanie
